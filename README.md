# USB HUB

- 2 USB-C 2.0 ports
- 2 USB-A 3.1 gen 1 ports
- Using TUSB8042, USB-A upstream
- ~80x54mm, all recepticles


- Need USB-A to ~ to connect with computer
- Only for data, please do not try to use for charging!

Made this because I needed a cheap and small USB hub I could carry so that my main hub could be kept at home.
